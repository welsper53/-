함수
- 자주 쓰는 실행 코드들을 블록({,지변성격}) 단위로 묶어서 패키징한 상태
- 자바와 다른점 : 파리미터로 함수로 넘길 수 있다
- 자주 쓰는 코드들을 기능 단위로 재사용하기 위함

함수 선언
- function 예약어를 이용

함수 호출
- 정의된 함수를 호출해야 기능이 실행된다

함수 종류
1. 선언적 함수
- 이름을 가짐
- 자바스크립트 파일을 읽을 때 선언적 함수를 가장 먼저 읽는다
    (자바의 static과 비슷)
- 선언적 함수 호출 위치가 자유로움
2. 익명 함수
- 함수의 이름이 없는 형태
    (-> 콜백 함수)
- 자체만으로 호출 불가
- 변수에 익명함수를 대입 또는 특정 이벤트 객체에 대입하는 형태로 호출
3. 대입 함수
- 변수에 익명함수가 대입된 형태
4. 즉시 실행 함수
- 카카오API(지도 서비스 코드)
